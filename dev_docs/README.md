## データの信頼性について

### 高信頼性データ（公的調査、公式URL確認済み）
以下のデータは、公的調査や公式レポートに基づいており、出典URLが確認済みです：
- ✅ IDEATECH「大企業・エンプラの意思決定プロセス実態調査」（2025年6月）
- ✅ スーパーソフトウエア「新規事業に関する実態調査」（2023年6月）
- ✅ PwC Japan「新規事業開発の取り組みに関する実態調査2025年」（2025年11月）
- ✅ NTTデータ「なぜ81％の新規事業がグロースに失敗するのか？」（2024年9月）
- ✅ パーソル総合研究所「無駄な社内会議による企業損失額調査」（2018年9月）

### 中信頼性データ（推定値）
一部のデータは、関連調査は確認済みですが、具体的な数値の出典URLが未確認のため推定値として扱っています：
- ⚠️ アルテリックス・ジャパン：戦略的意思決定平均20日
- ⚠️ 商社事例：年間1億2300万円の機会損失削減

### 低信頼性データ（フェルミ推定による計算値）
以下のデータは、フェルミ推定による計算値であり、推定の前提条件が含まれています：
- ⚠️ Zombie Project化率：35%
- ⚠️ 経済効果の絶対額：年間100-120億円規模

**詳細な出典情報とURLは、`Data/Helm_フェルミ推定による効果試算.md`を参照してください。**
