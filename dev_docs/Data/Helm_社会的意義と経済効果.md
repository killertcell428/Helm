## 1. 社会的意義の訴求

### 1.1 大企業の意思決定遅延による経済損失

#### 現状の課題（実データベース）

- **日本企業の意思決定遅延による経済損失**
  - 意思決定リードタイム：平均70-85日
    - 戦略的意思決定：20日（アルテリックス・ジャパン推定値）
    - 新規事業：120-150日（IDEATECH「大企業・エンプラの意思決定プロセス実態調査」2025年6月）
    - **出典URL：** https://prtimes.jp/main/html/rd/p/000000421.000045863.html
  - 新規事業の失敗率：70%（複数の調査結果の平均）
    - スーパーソフトウエア（2023年6月）：60%以上
    - PwC Japan（2025年11月）：投資回収まで20%
    - NTTデータ（2024年9月）：81%が失敗
    - **出典URL：**
      - https://prtimes.jp/main/html/rd/p/000000023.000004742.html
      - https://www.pwc.com/jp/ja/knowledge/thoughtleadership/new-business-development-survey-2025.html
      - https://www.nttdata.com/jp/ja/trends/event/archive/2024/097
  - 意思決定遅延による機会損失：年間**約20億円規模**（大企業、従業員580名の事例をスケールアップ）
    - 商社事例：決断力向上研修により機会損失を80%削減
    - **出典URL：** https://www.kensyu-mitsumori.com/cms/archives/898
    - **信頼性：中**（具体的な「年間1億2300万円」という数値の出典URL未確認）

- **Zombie Projectによる無駄な投資**
  - 大企業の新規事業プロジェクト数：平均70件/年（予算規模からの推定）
  - Zombie Project化率：35%（失敗プロジェクトのうち継続されている割合の推定）
    - **信頼性：低**（フェルミ推定による計算値）
  - Zombie Projectによる無駄な投資：年間**約123億円規模**（70件×35%×5,000万円/件）

#### Helm導入による効果（フェルミ推定）

- **意思決定速度の改善**
  - 意思決定リードタイム：70-85日 → 2日以内（**97%削減、35倍改善**）
  - 新規事業の失敗率：70% → 50%以下（**29%改善**）
  - 機会損失の削減：年間**約19億円規模の削減**（20億円→1億円以下）

- **Zombie Projectの早期検知**
  - Zombie Project検知率：0% → 90%以上
  - Zombie Project化率：35% → 10%以下（**71%削減**）
  - 無駄な投資の削減：年間**約88億円規模の削減**（123億円→35億円）

### 1.2 具体的な社会課題との接続

**Helm導入後：**

- 新規事業の失敗率が30%以上改善
- 雇用機会の創出：年間**数万人規模**

**Helm導入後：**
- Zombie Project検知率が0%から90%以上に向上
