# 優勝記事作成のための分析資料

## 📖 このディレクトリについて

過去のAI Agent Hackathon受賞記事（vol2、vol3）を分析し、HelmプロジェクトのZenn記事を最優秀賞レベルに仕上げるための資料をまとめています。

---

## 🎯 まずはこれを読む

### **優勝記事作成ガイド.md** ⭐最重要

**これがメインファイルです。まずはこれを読んでください。**

- 審査基準（課題の新規性、解決策の有効性、実装品質と拡張性）
- 過去の受賞記事分析結果（vol2 + vol3）
- 審査員情報（参考程度）
- 完全チェックリスト

このファイル1つで、優勝できる記事を作成するために必要な情報がすべて揃っています。

### ステップ1: メインファイルを読む

## ✅ チェックリスト

記事を作成する際は、**`優勝記事作成ガイド.md`** のチェックリストを参照してください。

### 【課題の新規性】
- [ ] 既存領域との対比
- [ ] 定量データで裏付け
- [ ] 権威ある資料の引用
- [ ] ストーリー形式
- [ ] 感情的な共感
- [ ] 具体的な数字

### 【解決策の有効性】
- [ ] 設計思想の明確化
- [ ] 実証実験の実施
- [ ] 定量的な成果
- [ ] Before/Afterのストーリー形式
- [ ] 技術的な実装詳細
- [ ] 具体的な利用フロー

### 【実装品質と拡張性】
- [ ] 技術スタック表
- [ ] 設計判断のプロセス
- [ ] AIエージェントの処理フロー図
- [ ] パフォーマンス最適化
- [ ] 拡張性の提示

### 画像使用
- [ ] デモ動画を冒頭に配置（必須）
- [ ] システムアーキテクチャ図を必ず含める（必須）
- [ ] ストーリー形式のイラスト
- [ ] 統計データの図表
