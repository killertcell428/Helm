## ドキュメントの目的

これらのドキュメントは以下の目的で作成されました：

1. **コンテンツの記録**: 各ページに表示されている内容をテキスト形式で記録
2. **設計の理解**: ページ構成や情報構造を理解するためのリファレンス
3. **コンテンツ管理**: 今後のコンテンツ更新や修正の際のベースライン
4. **ドキュメント化**: プレゼンテーションや説明資料作成時の参考資料
