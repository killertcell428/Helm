# デモ一覧ページ（Demo List Page）

## 概要
Helmのデモケース一覧を表示するダッシュボードページ。新規事業プロジェクトの状況を可視化し、各デモケースへの導線を提供する。

## ページ構成

### 1. ナビゲーションバー
- **Helmロゴ**: 左側にCompassアイコンとHelmブランド名
- **戻るボタン**: 「ホームに戻る」ボタン（/へリンク）

### 2. プロジェクトダッシュボードセクション
- **バッジ**: 「New Business Project」
- **タイトル**: 「新規事業プロジェクト ダッシュボード」
- **説明**: Helmによる組織構造の観測と最適化がリアルタイムで進行中

#### メトリクスグリッド（4つのカード）
1. **アクティブタスク: 8**
   - 前週比: +2
   - アイコン: Clock

2. **要判断アラート: 3**
   - Helmが判断を待機中
   - アイコン: AlertCircle（赤色）

3. **構造改善率: 87%**
   - 今月: +12%
   - アイコン: CheckCircle（緑色）

4. **参加ロール数: 12**
   - 内訳: 3 Executive, 5 Manager, 4 Staff
   - アイコン: Users

### 3. タスク一覧
- **タイトル**: 「タスク一覧」
- **フィルター**: ロール別フィルター（全て/Executive/Manager/Staff）

#### タスクカード（5件）
1. **経営会議後の構造変化検知**
   - ステータス: 要判断（alert）
   - 優先度: 高（high）
   - ロール: Executive, Manager
   - 説明: 「判断集中」「正当化フェーズ移行」の兆候を検知
   - 最終更新: 2時間前
   - リンク: /demo/case1

2. **重大ミスのエスカレーション**
   - ステータス: 要判断（alert）
   - 優先度: 緊急（critical）
   - ロール: Executive, Staff
   - 説明: 報告遅延・レビュー権限の構造的欠如
   - 最終更新: 30分前
   - リンク: /demo/case2

3. **外部トレンドのチャンス検知**
   - ステータス: 要判断（alert）
   - 優先度: 高（high）
   - ロール: Executive, Manager
   - 説明: 市場環境の急激な変化と既存戦略の乖離
   - 最終更新: 1時間前
   - リンク: /demo/case3

4. **顧客ヒアリング結果の構造化**
   - ステータス: 進行中（in-progress）
   - 優先度: 中（medium）
   - ロール: Manager, Staff
   - 説明: 15社のヒアリング内容から仮説を抽出中
   - 最終更新: 4時間前

5. **競合分析レポートの統合**
   - ステータス: 完了（completed）
   - 優先度: 中（medium）
   - ロール: Manager
   - 説明: 3社の競合分析が完了し、差別化ポイントを特定
   - 最終更新: 1日前

### 4. 過去7日間のHelm活動
- **タイトル**: 「過去7日間のHelm活動」
- **アイコン**: TrendingUp

#### 活動タイムライン（棒グラフ形式）
- **今日**: 10件（アラート: 25%, 進行中: 50%, 完了: 25%）
- **昨日**: 12件（アラート: 20%, 進行中: 40%, 完了: 40%）
- **3日前**: 11件（アラート: 15%, 進行中: 45%, 完了: 40%）
- **4日前**: 9件（アラート: 10%, 進行中: 50%, 完了: 40%）
- **5日前**: 14件（アラート: 30%, 進行中: 35%, 完了: 35%）

#### 凡例
- 要判断アラート（赤色）
- AI自律実行中（プライマリ色）
- 完了・継続観測（緑色）

### 5. 詳細デモケースセクション
- **タイトル**: 「詳細デモケース」
- **バッジ**: 「Interactive Demo」
- **説明**: 各ケースで、データ受領 → Helm解析 → ロール別判断 → AI自律実行 → 結果受領という完全なPDCAフローを体験できます

#### Case 1: 経営会議後の構造変化検知
- **検知内容**: 「判断集中」「正当化フェーズ移行」の兆候
- **Helmの提案**: 撤退条件の明文化とロール再設計
- **関与ロール**: Executive / Manager / Staff
- **リンク**: /demo/case1

#### Case 2: 重大ミスのエスカレーション
- **検知内容**: 報告遅延・レビュー権限の構造的欠如
- **Helmの提案**: 即座にExecutiveへエスカレーション
- **結果**: 分析担当をレビュー権限付きに追加
- **リンク**: /demo/case2

#### Case 3: 外部トレンドのチャンス検知
- **検知内容**: 市場環境の急激な変化と既存戦略の乖離
- **Helmの提案**: 緊急の戦略レビューとピボット案の作成
- **結果**: 48時間以内に新方針を決定し機会損失を防止
- **リンク**: /demo/case3

## インタラクション
- ロールフィルターでタスク一覧を絞り込み可能
- タスクカードをクリックで各デモケースページへ遷移
- 各デモケースカードの「デモ開始」ボタンで詳細ページへ遷移

## デザイン要素
- カードベースのレイアウト
- ステータス別の色分け（要判断: 赤、進行中: プライマリ、完了: グレー）
- グラデーション背景
